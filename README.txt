Инструкция использования:
1. Настройка (Config файл должен лежать в папке с программой)
1.1. В настройках указать путь к компилятору g++.
1.2. Задать ограничения.
1.3. Задать оценки к тестам.
2. Использование
2.1. Выбрать файл с кодом (компиляция и тестирование без выбранного файла работать не будут)
2.2. Нажать кнопку скомпилировать. (если код уже был скомпилирован, этот шаг не обязателен)
2.3. Выбрать в левом окне папку с тестами (среднее окно отображает содержимое)
2.4. Нажать кнопку тестировать.
